## Deploy to Render.com (free, 2 minutes)

1. Go to **render.com** and sign up (free)
2. Click **New +** → **Static Site**
3. Choose **"Deploy without a Git repository"** or drag-and-drop the whole `shiktra-portfolio` folder
4. Set **Publish directory** to `./`
5. Click **Create Static Site**
6. Render gives you a URL like `shiktra-portfolio.onrender.com` — that's your live site

**Alternative: Netlify (even simpler)**

1. Go to **app.netlify.com/drop**
2. Drag and drop the whole `shiktra-portfolio` folder onto the page
3. Done. You get a live URL immediately.

You can add a custom domain (e.g. `shiktra.com`) later on either platform once you buy one.
